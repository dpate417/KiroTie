ECONOMICS FRAME — REQUIRED GUARDRAIL

The Transparency Gate

Every prediction must be grounded in verifiable data. If not — it's blocked.

✓
Event type provided
✓
Signup count ≥ 1
✓
Planned quantity ≥ 1
✓
Valid event category
✓ APPROVED — Prediction grounded in USDA · EPA data
VS
✓
Event type provided
✗
Signup count missing
✗
Planned quantity = 0
✓
Valid event category
🚫 BLOCKED — Cannot produce data-backed estimate
# engine/gate.py
def transparency_gate(data):
  missing = [f for f in REQUIRED_FIELDS if not data.get(f)]
  if missing: return {"status": "BLOCKED"}
  return {"status": "APPROVED"}

ECONOMICS FRAME — REQUIRED GUARDRAIL

The Transparency Gate

Every prediction must be grounded in verifiable data. If it's not — it's blocked.

Input Data
✅ Event type provided
✅ Signup count ≥ 1
✅ Planned quantity ≥ 1
✅ Valid event category
↓
✅ APPROVED
Prediction grounded in USDA · EPA data
VS
Input Data
✅ Event type provided
❌ Signup count missing
❌ Planned quantity = 0
✅ Valid event category
↓
🚫 BLOCKED
Missing required fields — cannot produce data-backed estimate
# engine/gate.py — Transparency Gate
